23 /// Interface of the function that translates the sources managed by `sourceMgr`
24 /// to MLIR. The source manager has at least one buffer. The implementation

30 /// Interface of the function that translates the given string to MLIR. The

42 /// Interface of the function that performs file-to-file translation involving
43 /// MLIR. The input file is held in the given MemoryBuffer; the output file
44 /// should be written to the given raw_ostream. The implementation should create
45 /// all MLIR constructs needed during the process inside the given context. This
46 /// can be used for round-tripping external formats through the MLIR system.
47 using TranslateFunction = std::function<LogicalResult(
48  llvm::SourceMgr &sourceMgr, llvm::raw_ostream &output, MLIRContext *)>;
